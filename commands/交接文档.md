# 生成交接文档

用户调用了 `/交接文档`。

请执行与 `/handoff` 完全相同的流程：

1. 读取本插件的 `commands/handoff.md`。
2. 按其中的要求检查当前工作区。
3. 在当前工作区根目录生成 `HANDOFF.md`。
4. 尽可能创建新的 Codex 对话，并把 `HANDOFF.md` 的内容或绝对路径放入新对话初始提示中。
5. 如果当前环境无法自动点击 Codex 原生界面或自动导入附件，请明确说明限制，不要假装已经完成 UI 点击或附件导入。

最终必须保存文件名：`HANDOFF.md`。
